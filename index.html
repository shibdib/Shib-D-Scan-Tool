    Shib-D-Scan-Tool

    EVE Online Dscan Parser Built around the now depreceated https://github.com/recon-null/dscan_reporter

    Current as of 20March2015

    Example Site - http://shibdib.com/ds/1/

    Installation

    • Clone to an apache or NGINX server
    • Use the db.sql file to make your mysql tables
    • Modify config.php for your DB
    • Enjoy :)

    Dependancies

    • SQL DB
    • A Web Server
